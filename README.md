# Solvefy Interview Evaluation Portal (Static Bundle)

This is a **static, client-side** HR interview evaluation tool:

- MCQ-first questions with optional notes
- Weighted scoring and **A+ → F** categorization
- **Decision Matrix Alignment** by dimension (table + chart)
- **Download JSON** (ATS-ready) and **Print/Save PDF**

## Files
- `index.html` – single-page app
- `logo.svg` – header branding

## Run locally
Open `index.html` in any modern browser.

## Deploy (GitHub Pages)
1. Create a new GitHub repo (public or private).
2. Upload `index.html` and `logo.svg` to the repo root.
3. In repo settings → Pages → Set source to **main** branch, **/ (root)**.
4. Visit the Pages URL (e.g., `https://<org-or-user>.github.io/<repo>/`).

## Deploy (Vercel)
1. Install Vercel CLI, or use Vercel dashboard.
2. Create a new project from this folder.
3. Framework preset: **Other** (no build step needed).
4. Deploy – your URL will be live immediately.
